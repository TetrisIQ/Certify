Verifiziere dein Zeugnis

Folge der Schritt für Schritt anweisung

  • Scanne den QR-Code auf dem Dokument
  • Überprüfe die Noten und Fächer
  • Stimmt alles überein, starte die Überprüfung
  • Vergleiche den code den du bekommen hast mit dem VerifyToken auf dem Dokument
  • Stimmt der Code überein, stimmen die daten überein

Wie Funktioniert die Verifizierung

Einfach erklärt

Für die Verifizierung muss die Organisation die dir das Zeugnis ausgestellt hat, dieses hier zuerst Registrieren. Auf deinem Zeugnis sollte ein QR-Code und ein VerifyToken zu sehen sein.

Zum Verifizieren musst du lediglich den QR-Code scannen, die angegebenen Daten mit dennen auf dem Zeugnis vergleichen und das VerifyToken genau überprüfen.

Wenn alles übereinstimmt ist das Zeugnis Verifiziert!

Ein Technischer blickwinkel

Bei der Regestrierung des Zeignisses wird von den daten, in dem Zeugnis eine Hash summe (sha512) gebildet. Diese wird mit dem VerifyToken im Backend gespeichert.

Es wird nicht das gesamte zeugnis gespeichert, sondern nur dieser Hash wert und das Token. Bei der Verifizirung wird der selbe Algorythmus verwendet und es wird nur die Hash summe abgeglichen.

Im QR-Code werden die Zeugnisinforationen über Fächer und Noten gespeichert und beim scannen an die Webseite übermittelt (aber nicht gespeichert).

Dies ist ist vergleichbar mit dem Mauellen eintragen aller noten, nur das der QR-Code schneller ist und keine fehler beim Abtippen auftreten.
